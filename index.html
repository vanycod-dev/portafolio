Acerca de mí

Hola, si llegaste hasta aquí es porque buscas a alguien que te ayude con el desarrollo de algún proyecto o, si ya cuentas con uno, necesitas una mano adicional para implementar soluciones en equipo. Ya sea en grupos pequeños o grandes, con poco tiempo o un calendario ajustado, puedo ayudarte. Aunque no tengo un trabajo formal en el ámbito del desarrollo de software, sí cuento con experiencia en entornos donde el tiempo es limitado y se requieren soluciones rápidas para alcanzar metas. Si continúas bajando, verás mi experiencia en desarrollo, así como las tecnologías que manejo.

Gestor de Metas

Gestor de Metas Progresivas - Aplicación web SPA (Single Page Application) desarrollada con JavaScript vanilla que permite crear, editar y eliminar objetivos personales, incluyendo una barra de progreso interactiva que se actualiza automáticamente. Utiliza localStorage para persistencia de datos sin necesidad de backend, con interfaz responsive construida con HTML5 semántico y CSS3 moderno.

Proyecto 3

Aplicación completa que te permite crear notas tanto de manera local como en la nube, para que puedas acceder a ellas desde cualquier lugar con solo tu usuario y contraseña. Desarrollé tanto el frontend como el backend y la base de datos, utilizando servidores diferentes pero conectados entre sí. Tecnologías implementadas: Frontend: HTML, CSS, JavaScript, React, React Router y Tailwind CSS,Backend: Node.js con Express, Base de datos: MySQL
